# 개요

> [!important] 요약
> 본 문서는 게임 내 무기 아이템의 기본 시스템, 스탯, 타겟팅 방식, 세부 무기 목록 및 진화 시스템을 포함한 무기 관련 제반 기획 요소를 정의한다.

> [!important] 주요 특징
> - 완전 자동 공격 시스템 (플레이어 조작 배제)
> - 무기별 고유 쿨타임 및 발동 메커니즘 보유
> - 투사체, 범위 공격 등 다양한 공격 타입 존재
> - 특정 패시브 아이템과 조합 시 무기 진화 가능
> - 최대 6개(확장 시 +1) 무기 동시 장착 지원

# 무기 시스템

## 기본 개념

> [!tip] 자동 공격 시스템
> - 플레이어의 직접 공격 조작 불가 (공격 키 미존재).
> - 장착 무기의 **공격 속도** 스탯에 따라 **자동 발동**.
> - 각 무기는 **독립적인 쿨타임**을 가지며, 동시 장착 시 개별 발동.
> - 무기별 고유 타겟팅 및 발동 알고리즘(투사체, 범위 등) 적용.

## 무기 획득 및 강화

> [!info] 무기 획득
> - **레벨업 선택지**를 통해 신규 무기 획득 또는 기존 무기 강화.
> - 무기 슬롯에 여유가 있을 경우에만 신규 무기 등장.
> - 무기 슬롯: **기본 6개** ("확장 모듈" 패시브 습득 시 +1 확장 가능).

> [!example] 무기 강화
> - 강화 단계: **Lv.1 ~ Lv.8**.
> - 레벨업 선택지에서 해당 무기 선택 시 강화 적용.
> - 강화 시 무기 스탯(피해량, 공격 속도, 투사체 개수 등) 상승.
> - Lv.8 (최대 레벨) 도달 및 특정 조건 충족 시 진화 가능.

## 무기 공통 스탯

> [!tip]
> 모든 무기는 하단 스탯으로 구성되며, 무기별 고유 수치를 보유함.

| 항목 | 자료형 | 설명 |
| :--- | :--- | :--- |
| **기본 피해량** | int | 무기의 기본 피해량. (int 형변환) |
| **기본 치명타 확률** | float | 치명타 발동 확률 (%). (0~100 범위)<br>소수점 제2위 반올림 적용. |
| **기본 치명타 배수** | float | 치명타 발동 시 피해량 배율. (예: 2.0 = 2배 피해)<br>소수점 제2위 반올림 적용. |
| **기본 공격 속도** | float | 공격 간 쿨타임.<br>소수점 제2위 반올림 적용. |
| **기본 공격 범위** | float | 공격 사거리.<br>소수점 제2위 반올림 적용. |
| **기본 투사체 속도** | float | 투사체(총알, 화살 등) 비행 속도. (단위: units/sec)<br>소수점 제2위 반올림 적용. |
| **기본 투사체 개수** | int | 1회 공격 시 발사되는 투사체 수량. (예: 3 = 동시 3발 발사) |

> [!note] 참고
> 무기 레벨 상승 시 해당 스탯 강화 적용. 세부 수치는 밸런스 테스트를 통해 조정됨.

## 타겟팅 방식

> [!tip]
> 각 무기는 고유한 사정거리 스탯을 가지고 있으며, 그 사정거리 스탯 안에 들어온 적들 중 가장 가까운 적을 우선 공격한다.
