## 1. 적 기본

### 개요

- 모든 적은 HP와 이동 속도, 공격력, 시야를 스탯으로 가진다.
- 이 중 공격력은 모든 적이 고정된 수치(적이 플레이어에게 입히는 피해량은 같음)

### 타겟 선정 방식

> 기본적으로 아지트를 향해 직선으로 이동

**기본 타겟:** 아지트

**시야 범위 내 감지 시 타겟 변경**:

1. 적의 시야 범위 내에 플레이어 또는 포탑이 들어오면 해당 대상으로 타겟 변경
2. 여러 대상이 시야에 들어오면 **가장 가까운 대상**을 우선 공격
3. 적이 대상에게 닿으면 공격 판정

적 종류 중엔 플레이어나 포탑을 무시하고 아지트만을 향해 돌진하는 적도 존재

## 2. 적 스폰

### 스폰 위치

맵 8방향 가장자리에 스폰 위치가 있다.

### 웨이브 시스템

> ⏱️ 시간 기반 난이도 상승
> 
> 게임은 명시적인 "웨이브 1, 2, 3..."을 표시하지 않지만, 내부적으로는 시간대별로 등장하는 적의 종류와 수가 정해져 있다.

**시간대별 스폰 테이블 예시**:

| 시간 (분:초)     | 등장 적 종류       | 스폰 개체 수 |
| ------------ | ------------- | ------- |
| 0:00 - 1:00  | 극초반 적         | 낮음      |
| 1:00 - 2:40  | 극초반 적 + 초반 적  | 중간      |
| 2:40 - 5:30  | 초반 적 + 빠른 적   | 높음      |
| 5:30 - 8:30  | 모든 일반 적 + 돌격형 | 높음      |
| 8:30 - 10:00 | 모든 적          | 매우 높음   |

**스폰 빈도**:

- 초반: 10-15초마다 소규모 그룹 스폰
- 중반: 8-10초마다 중규모 그룹 스폰
- 후반: 5-8초마다 대규모 그룹 스폰

## 3. 적 종류

### 3.1 일반 적

### 🤖 소형 로봇

> 🤖 타입: 기본형
> 
> **설명**: 극초반에 나오는 가장 기본이 되는 적. 느리지만 꾸준히 아지트를 향해 진격한다.

| 스탯    | 수치            |     |
| ----- | ------------- | --- |
| HP    | 1             |     |
| 이동 속도 | 2.0 units/sec |     |
| 시야    | 5 units       |     |
| 드랍    | 경험치 (소량)      |     |


**특징**:
- 게임의 가장 초반에 등장

---

### 🤖 중형 로봇

> 🤖 타입: 기본형
> 
> **설명**: 약간 업그레이드 된 적. 느리지만 꾸준히 아지트를 향해 진격한다.

| 스탯    | 수치            |                                         |
| ----- | ------------- | --------------------------------------- |
| HP    | 15            |                                         |
| 이동 속도 | 2.5 units/sec | 공격력이 낮은 무기는 2~3대 높은 무기는 1대에 처치 가능하도록 조정 |
| 시야    | 5 units       |                                         |
| 드랍    | 경험치 (소량)      |                                         |

**특징**:
- 게임의 초반에 등장

---

### ⚡ 빠른 스카우트

> ⚡ 타입: 고속형
> 
> **설명**: 빠른 속도로 아지트에 접근하는 위협적인 적. 낮은 HP로 쉽게 처치할 수 있지만 놓치면 금방 아지트에 도달한다.

| 스탯    | 수치            |
| ----- | ------------- |
| HP    | 1             |
| 이동 속도 | 4.5 units/sec |
| 시야    | 6 units       |
| 드랍    | 경험치 (소량)      |

**특징**:
- 매우 빠른 이동 속도
- 포탑이나 플레이어가 빠르게 대응해야 함
- 집중 공격이 필요한 우선 타겟

**비주얼**: 날렵한 형태, 청록색 잔광 효과

---

### 🛡️ 사이보그

> 🛡️ 타입: 탱커형
> 
> **설명**: 게임 후반에 나오는 느리지만 매우 높은 HP를 가진 적. 많은 화력을 집중해야 처치할 수 있다.

| 스탯    | 수치            |
| ----- | ------------- |
| HP    | 50            |
| 이동 속도 | 1.0 units/sec |
| 시야    | 4 units       |
| 드랍    | 경험치 (중량)      |

**특징**:
- 높은 HP
- 느린 이동 속도
- 관통 공격이나 폭발 공격에 효과적

**비주얼**: 거대하고 무거운 기계, 금속 장갑

---

### 🏃 미치광이

> 🏃 타입: 돌격형
> 
> **설명**: 플레이어와 포탑을 완전히 무시하고 오직 아지트만을 향해 돌진하는 적. 매우 빠르고 직선으로 달려온다.

| 스탯    | 수치             |
| ----- | -------------- |
| HP    | 50             |
| 이동 속도 | 5.0 units/sec  |
| 시야    | 없음 (타겟 변경 안 함) |
| 드랍    | 경험치 (중량)       |

**특징**:
- **시야 무시**: 플레이어와 포탑을 인식하지 않음
- 아지트만을 타겟으로 설정하고 절대 변경하지 않음
- 매우 빠른 속도
- 플레이어가 직접 막거나 경로상에 포탑 배치 필요

**비주얼**: 날카롭고 빠른 형태, 잔광 효과
