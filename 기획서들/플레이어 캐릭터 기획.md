# 1. 개요

## 요약

이 문서는 게임 내 플레이어 캐릭터의 기본 스탯, 이동 시스템, 전투 시스템, 사망 및 부활 메커니즘, 레벨 시스템을 포함한 플레이어 관련 모든 기획 요소를 다룹니다.

## 주요 특징

- WASD/방향키를 이용한 8방향 자유 이동
- 조작 키가 없는 완전 자동 공격 시스템
- 경험치 기반 레벨업과 랜덤 보상 선택
- 무기/패시브 슬롯 제한을 통한 전략적 빌드 구성
- 특정 조건 충족 시 무기 진화 시스템

# 4. 이동 시스템

## 특징 및 조작법

- **조작:** WASD 또는 방향키를 통한 8방향 자유 이동.
- **8방향 이동:** 대각선 방향 이동 가능 (속도는 정규화하여 일정하게 유지).
- **충돌 처리:** 아지트, 포탑 등의 구조물은 통과할 수 없음.
- **주요 조작:** 캐릭터의 주 조작은 이동으로만 제한됨.

## 애니메이션 및 스프라이트

- **Idle 상태:** 이동 버튼 입력이 없을 때 적용.
- **Run 상태:** 이동 버튼 입력이 있을 때 적용.
- **좌우 반전 조건:**
    - 기본 이미지는 오른쪽을 향함.
    - 왼쪽 이동 시 이미지를 뒤집어(Flip) 표현.
    - 이동 중지 시 마지막으로 바라보던 방향 유지.

# 5. 전투 시스템

## 자동 공격
- 플레이어는 직접 공격 조작을 하지 않음.
- 장착한 무기가 각 무기의 '공격 속도' 스탯에 따라 자동으로 발동.
- 무기마다 고유한 타겟팅 방식(가까운 적, 무작위 등)과 발동 방식(투사체, 범위 등)을 가짐.

## 피격 시스템
- **정의:** HP가 어떠한 방식으로든 줄어드는 상황을 '피격'으로 정의.
- **피격 이펙트:**
    - **비주얼:** 에셋으로 제공되는 피격 애니메이션 재생.
    - **사운드:** (추후 선정)
    - **데미지 표시:** 수치는 표시하지 않고, UI의 체력 게이지 감소로만 직관적으로 알림.

## 데미지 계산 공식
최종 피해량은 플레이어의 방어력에 따라 감쇄된다.

$$최종 피해량 = round \left( 적 공격력 \times \frac{100 - 방어력}{100} \right)$$

# 6. 사망 및 부활 시스템

## 플레이어 사망
- **조건:** HP가 0 이하가 될 경우.
- **사망 프로세스:**
    1. 사망 애니메이션 재생
    2. 애니메이션 종료 후 게임 오브젝트 삭제
    3. 부활 시스템 작동 (목숨 -1)
    4. 부활 가능 시 아지트 위치에서 재생성

## 부활 시스템
- **부활 메커니즘:**
    - **부활 위치:** 아지트 중앙
    - **부활 시간:** 사망 후 5초 뒤 자동 부활
    - **부활 시 상태:** 최대 HP로 회복, 2초간 무적 상태 유지
- **부활 조건:**
    - 목숨 1 이상: 부활 가능
    - 목숨 0: 부활 불가, 게임 오버

# 7. 레벨 시스템

## 레벨 구조
- **시작 레벨:** 1레벨
- **최대 레벨:** 99레벨 (실질적으로 무제한)
- **필요 경험치:**
    - 1~10레벨: 100 exp / 레벨
    - 11~20레벨: 150 exp / 레벨
    - 21~30레벨: 200 exp / 레벨
    - 31레벨 이상: 250 exp / 레벨

## 경험치 획득
- **경험치 오브젝트:** 적 처치 시 드랍되는 파란색 보석. 일정 범위 내 접근 시 자동 흡수.
- **적 유형별 경험치:** 일반 적(5), 정예 적(15), 보스 적(50).
- **보너스:** 연속 처치(콤보) 시 경험치 +10% 추가 획득.

## 레벨업 시 변화
- **보상 선택:** 3개의 무작위 선택지 제공 (10레벨 단위에서는 4개 제공).
- **선택지 구성 확률:**
    - **슬롯 여유 시:** 신규 무기(30%), 신규 패시브(30%), 기존 강화(30%), 포탑 업그레이드(10%)
    - **슬롯 가득 참:** 기존 아이템 강화(70%), 포탑 업그레이드(30%)
- **리롤 기능:** 50G 소모하여 선택지 재생성 (레벨당 1회).

## 아이템 슬롯 및 무기 진화
- **슬롯 제한:** 무기 6개, 패시브 6개 기본 제공.
- **무기 진화:**
    - **조건:** 특정 무기 Lv.8 도달 + 조합되는 특정 패시브 아이템 보유.
    - **효과:** 성능 대폭 강화, 비주얼 변화, 이름 변경.

## UI 및 연출(Feel)
- **UI:** 화면 상단 경험치 바, 레벨 표시(Lv.XX), 다음 레벨까지 남은 경험치 표시.
- **연출:** "LEVEL UP!" 텍스트, 섬광 효과, 카드 펼치기 애니메이션, 보상 선택 시 강조 효과.
